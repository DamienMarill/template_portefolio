À propos de moi

Ohayo gozaimasu ! Je suis Neko-chan, un développeur web avec le power level d'un Saiyan. Comme Tanjiro dans Demon Slayer, j'ai affiné mes techniques de développement jusqu'à pouvoir trancher n'importe quel bug d'un seul coup de clavier !

Mon ambition ? Créer des sites web aussi époustouflants que les transformations de Sailor Moon, aussi rapides que Sonic the Hedgehog, et aussi robustes que l'armure d'Alphonse Elric. Prêt à rejoindre mon équipage pour des aventures digitales dignes de One Piece ?

Mes Projets Légendaires

Ninja Code

Un IDE révolutionnaire qui permet de coder aussi rapidement que Naruto forme ses sceaux. Avec des fonctionnalités comme le "Shadow Clone Debugging" et le "Rasengan Refactoring", vos bugs n'ont qu'à bien se tenir !

Mecha Responsive

Un framework CSS inspiré des mechas de Neon Genesis Evangelion. Vos sites web s'adapteront à tous les écrans aussi facilement qu'un EVA s'adapte à son pilote. Attention aux berserk mode sur les petits écrans !

Ghibli Task

Une application de gestion de tâches magique, où chaque tâche accomplie fait pousser un arbre dans votre jardin virtuel. Inspiré par la beauté des films du studio Ghibli, ce projet rend la productivité aussi enchanteresse qu'un voyage avec Totoro.

Pokédata

Une base de données interactive de tous les Pokémon, aussi complète que le Pokédex du Professeur Chen. Avec des graphiques de stats aussi colorés que les attaques de Pikachu, et un système de recherche plus rapide qu'un Rapidash !

Shonen Jump Start

Un générateur de landing pages qui crée des designs aussi accrocheurs que les couvertures du Weekly Shonen Jump. Vos visiteurs seront plus captivés que Zenitsu lorsqu'il dort !

Full Metal DevOps

Une suite d'outils DevOps qui automatise vos déploiements avec la précision d'une transmutation d'Edward Elric. Échangez l'équivalent de votre temps contre des déploiements parfaits !
