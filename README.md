# Portfolio Ultra Plus Ultra 🚀✨

Bienvenue dans le repo du Portfolio Plus Ultra ! Ce projet est un template de portfolio pour les développeurs web qui veulent montrer leurs compétences avec style, une touche d'humour et beaucoup de références aux animes.

## 🌟 Caractéristiques

- Design responsive adapté à tous les appareils (du smartphone de Senku au super-ordinateur de la Capsule Corp)
- Sections pour présenter vos projets, compétences et informations de contact
- Thème personnalisable via des variables CSS (aussi flexible que Luffy !)
- Intégration d'un logo personnalisé
- Structure HTML sémantique pour un meilleur référencement (SEO)
- Style moderne avec des effets de survol et des ombres

## 🚀 Démarrage

1. Clonez ce repo sur votre machine locale :
   ```
   git clone https://github.com/DamienMarill/template_portefolio.git
   ```
2. Ouvrez le fichier `index.html` dans votre navigateur préféré.
3. Admirez votre nouveau portfolio en action !

## 🛠 Personnalisation

### Contenu

1. Ouvrez `index.html` dans votre éditeur de code favori.
2. Remplacez le texte lorem ipsum par vos propres informations :
    - Mettez à jour la section "À propos" avec votre histoire.
    - Ajoutez vos propres projets dans la section "Projets".
    - Personnalisez la liste des compétences dans la section "Techniques Secrètes".
    - Mettez à jour vos informations de contact.

### Style

1. Ouvrez la balise `<style>` dans `index.html`.
2. Modifiez les variables CSS dans le sélecteur `:root` pour changer les couleurs et les espacements :
   ```css
   :root {
     --color-primary: #votre-couleur-principale;
     --color-secondary: #votre-couleur-secondaire;
     /* ... autres variables ... */
   }
   ```

### Logo

1. Remplacez l'URL du placeholder dans la balise `<img>` du header par l'URL de votre propre logo :
   ```html
   <img src="chemin/vers/votre-logo.png" alt="Votre Logo" class="logo">
   ```

### Images des projets

1. Remplacez les URLs des placeholders dans les balises `<img>` de la section projets par les URLs de vos propres images de projets.

## 📦 Déploiement

Pour déployer votre portfolio, vous pouvez utiliser des services comme GitHub Pages, Netlify, ou Vercel. Ces plateformes vous permettront de partager votre portfolio aussi facilement que Goku utilise son Instant Transmission !
